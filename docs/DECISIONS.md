# Architecture Decision Records (ADR)

This document contains decisions made regarding the technology stack, library choices, and architecture patterns.

## ADR-001: Technology Stack Selection

### Context
We need a robust, scalable, and type-safe stack for our logistics intelligence platform.

### Decision
- **Frontend**: Angular (for strong architecture, type safety, and rich features).
- **Backend**: Spring Boot (for robustness, security, and enterprise capabilities).
- **Database**: PostgreSQL / Relational DB.

### Status
Accepted
